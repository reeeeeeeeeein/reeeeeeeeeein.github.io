WSL(Windows Subsystem for Linux)可以在windows中装一个Linux子系统，非常方便且易于使用。

点开下载的界面会花很长时间，但下载速度很快。

下载过程中，打开Windows功能，勾选”适用于 Linux 的 Windows 子系统”:

然后打开下载好的Ubuntu，第一次打开要设定用户名和密码。这样，wsl就算是安装完了。
